# Simple Libev HTTP Server

## Description
Simple Libev HTTP Server that provides access to 2 different static pages with memory of 500B and 20KB.

## Installation Guide

### Server Installation
1. Go to the project folder
2. Enter the following command
```shell
$ make
```

### Running The Server
1. Go to the project folder
2. Enter the following command
```shell
$ ./server
```
